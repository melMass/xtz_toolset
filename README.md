## Getting Started

Edit utils/constants.js page to properly set the url of your own indexer ```HICDEX_MAIN```

Then run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
